# zenbun

## 概要
テキストファイルの全文検索ツールです。

* grepコマンドのように、ローカル端末上の指定フォルダ配下にあるテキストファイルに特定の文字列が含まれないか検索します。
  * インデックスは作成しません。  
* 検索条件ならびに検索結果はGUIを通して指定/表示します。

## 実装条件

* ビルドツール: Gradle 7.1
* Java: 8以上
* ソースコード: Groovy
  - テストコードを含む。
* GUI: Swing

## コマンド仕様
### 一時的なファイルを削除

以下のコマンドで一時的なファイルを削除します。

```
gradle clean
```

### テスト

以下のコマンドでテストを実行します。

```
gradle test
```

### 起動

以下のコマンドで検索ウィンドウを起動します。

```
gradle execute
```

## 画面仕様
### 検索ウィンドウ

* 検索条件指定ペイン
  - 検索条件を指定します
  - ウィンドウの上半分を占めます
* 検索結果表示ペイン
  - 検索結果を表示します
  - ウィンドウの下半分を占めます

### 検索条件指定ペイン

* 「ルートフォルダ」テキストフィールド
  - 検索対象となる最上位のフォルダのフォルダパスを指定します
  - デフォルト表示は空文字、最後に入力された値を保持します
* 「フォルダ指定」ボタン
  - ルートフォルダを指定するためのボタンです
  - ボタンクリックでフォルダ指定のためのOS標準のウィンドウが開きます  
    指定したフォルダはルートフォルダ欄に反映されます
* 「フォルダ履歴」ボタン
  - ボタンクリックでルートフォルダの履歴を参照するためのウィンドウが開きます  
    ダブルクリックで指定したキーワードがルートフォルダ欄に反映されます
* 「フォルダ履歴消去」ボタン
  - ボタンクリックでルートフォルダの履歴を消去します
* 「検索キーワード」テキストフィールド
  - 検索したいキーワードを指定します
* 「履歴」ボタン
  - ボタンクリックで検索キーワードの履歴を参照するためのウィンドウが開きます  
    ダブルクリックで指定したキーワードが検索キーワード欄に反映されます
* 「履歴消去」ボタン
  - ボタンクリックで検索キーワードの履歴を消去します
* 検索オプション
  * 「正規表現」チェックボックス
    - 検索キーワードを正規表現とみなすときチェックします  
      デフォルト表示はオフ、最後に選択した状態を保持します
  * 「大文字小文字を無視」チェックボックス
    - 検索キーワードについて大文字小文字を無視したいときチェックします  
      デフォルト表示はオフ、最後に選択した状態を保持します
* 「検索」ボタン
  * 検索を実行します  
    エンターキーでも実行します

## 検索結果表示ペイン

以下を表示します。  
検索結果をダブルクリックすると、コマンドを実行します。実行するコマンドは設定ファイルに設定します。

* 検索結果件数
* 検索結果
  * ファイルパス
    * ルートフォルダからの相対パスを表示します
  * 行番号
  * 列番号
  * 当該行
    - 検索キーワードが該当する箇所を太字で表示します
    - 長い行は右端で折り返して表示します

## ファイル仕様
### 設定ファイル

設定値を保持します。  
YAML形式です。  
以下のキーがあります。

* command
  - 検索結果をダブルクリックしたときに実行するコマンドを指定します。
  - 以下のパラメーターを埋め込むことができます。
    - `${filePath}`: ファイルパス（絶対パス）
    - `${lineNo}`: 行番号
    - `${columnNo}`: 列番号

### 状態保存ファイル

検索条件指定ペインの状態を保持します。  
YAML形式です。  
以下のキーがあります。

* rootFolder: ルートフォルダの入力値を保持
* historys: 検索キーワードの履歴をリストで保持
* option
  - regex: 正規表現か否か状態を真偽値で保持
  - ignoreCase: 大文字小文字を無視するか状態を真偽値で保持
