        • static Object getObject(Class<?> clazz)

          クラス名に対応する XMLを読みこんで、XML Schemaで検証し、バインディングしたインスタンスを返します。
          XML Schema検証にも、クラス名に対応するXML Schema定義ファイルを参照します。
          バインディングにも同じクラスを使用します。
          このメソッドはメンバ変数の宣言時に利用することを想定しているため、例外を投げません。

          Parameters:
          clazz - XMLファイルと XML Schema定義ファイル参照とバインディングのためのクラス
          Returns:
          XMLをバインディングしたインスタンス（検証エラー時は null）
